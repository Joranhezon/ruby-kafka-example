# Kafka Ruby Script Example

This is a simple Kafka example using ruby and the gem ```ruby-kafka```, used in a presentation made at [RubyConf Brasil 2021](https://online.rubyconf.com.br/). The purpose of this code and presentation is to introduce newcomers to Kafka and how to use it with Ruby.
